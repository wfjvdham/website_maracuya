Animations in Shiny using TweenLite

Jun 8, 2018 From a colleague I heard about TweenLite, a JavaScript library that can be used to animate objects on your website. You can move elements around by controlling their speed and position. I wanted to see how well this would work in a Shiny application so I experimented a little bit with that. The result is an application that can keep track of the score of the the weakest link, a TV game show from more than 10 years ago.

Colombian Cooking

May 23, 2018 Last December I had the opportunity to work with my friends from randommonkey. We made special about the Colombian kitchen. Part of this special was a Shiny application that can be used to search trough the Colombian recipes. Names, ingredient, region, complexity and the time required can be used to find your favorite one. After finding a recipe you can download the description. So if you are interested in making some Mojarras fritas (and you know a little Spanish because the page is completely in Spanish) look here.
